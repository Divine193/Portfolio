This dataset contains insurance poicyholder and claim information used to analyze factors influencing insurance claim approval dcisions.

### Data Source

The dataset was obtained from Kaggle and represents a simulated insurance claims dataset for analytical and educational purposes.

### Dataset Size

The dataset contains 10,000 records with 38 columns

### Key Columns:

  - `poiicy_id` : Unique identifier for each policy
  - `age` : Age of the policyholder
  - `education_level` : Highest education level  attained
  - `insurance_type` : Insurance type bought by the policyholder
  - `premium_amount` : Premium amount paid by the policyholder
  - `claim_amount` : Amount claimed by the policyholder
  - `any_injury` : indicates whether injuries occurred
  - `incident_state` : State where the incident occurred
  - `claim_status` : Claim outcome(Accepted or Denied)

The dataset containd missing values, inconsitent categorical entries, and class imbalance, which were handled during data cleaning and preprocessing.
